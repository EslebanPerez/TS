# Introducción a los tipos
### Strings
```
"Matias Perez"
'TypeScript'
`<h1>Hola Mundo</h1>`
```
### Números
```
pi = 3.141592
salario = 1500.00
entero = 1
```
### Booleanos
```
Verdadero(true)
Falso(false)
```
### Null y Undefined
```
numero = null
persona = undefined
```
### Objetos Literales
```
var persona = {
  nombre: "Evan"
  edad: 26
}
```
### Clases
```
class persona{
  nombre;
  edad;
}
```
### Funciones
```
function saludar(){
  return "Hola Mundo";
}
```

Se pueden crear nuevos tipos a los que ya trae TS por defecto
También se pueden crear interfaces

[Documentación](http://www.typescriptlang.org/docs/handbook/basic-types.html)
